Instalar la aplicación Itsas Lagunak en otro ordenador (demo)

Esta guía explica cómo montar una copia completa e independiente de la aplicación en un portátil Windows, con tus datos reales, para enseñarla. Tiempo aproximado: 1 hora. No hace falta saber programar: solo seguir los pasos y copiar/pegar los comandos.

Qué vas a necesitar: el portátil de demostración (Windows 10/11), una memoria USB (o Dropbox) para pasar archivos del ordenador principal al portátil, y conexión a internet en el portátil para descargar dos programas.
Importante — apunta esto antes de empezar. En el paso de PostgreSQL te pedirá inventar una contraseña para el usuario postgres. Apúntala en un papel, la necesitarás varias veces. En esta guía la llamaremos TU_CONTRASEÑA.

Parte 1 · Instalar los dos programas base en el portátil

1.1 — Node.js

  1. En el portátil, abre el navegador y entra en https://nodejs.org
  2. Descarga la versión LTS (el botón grande de la izquierda).
  3. Ejecuta el instalador y pulsa Siguiente / Next en todo (acepta los valores por defecto) hasta Finish.

1.2 — PostgreSQL 17

  1. Entra en https://www.postgresql.org/download/windows/ → "Download the installer" → elige la versión 17.
  2. Ejecuta el instalador. Pulsa Siguiente hasta llegar a la pantalla de contraseña.
  3. Escribe TU_CONTRASEÑA (la que apuntaste). Apúntala bien.
  4. Cuando pregunte el puerto, déjalo en 5432.
  5. Sigue pulsando Siguiente hasta el final. Si al terminar abre el "Stack Builder", ciérralo (no hace falta).
Comprobación rápida. Abre el menú Inicio, escribe cmd y pulsa Enter. En la ventana negra escribe node --version y pulsa Enter: debe salir un número (ej. v20.x). Eso confirma que Node está instalado.

Parte 2 · Copiar la aplicación del principal al portátil

NO copies la carpeta node_modules (es enorme y se vuelve a generar sola). Copia todo lo demás.
  1. En tu ordenador principal, ve a la carpeta del proyecto:
    C:\Users\User\Documents\Claude\Projects\Capturas
  2. Selecciona todo el contenido EXCEPTO la carpeta node_modules (y si existe, tampoco copies .next).
  3. Cópialo a la memoria USB.
  4. En el portátil, crea una carpeta sencilla, por ejemplo C:\Capturas, y pega ahí todo el contenido del USB.

Parte 3 · Copiar los datos reales para que la demo tenga contenido

Vamos a hacer una copia de la base de datos del ordenador principal y restaurarla en el portátil. Así la demo mostrará tus capturas, gastos y nóminas reales.

3.1 — Generar la copia (en el ordenador PRINCIPAL)

Abre cmd y pega este comando (te pedirá la contraseña de PostgreSQL de tu ordenador principal):

"C:\Program Files\PostgreSQL\17\bin\pg_dump.exe" -U postgres -d capturas -f "%USERPROFILE%\Desktop\capturas_datos.sql"

Se creará el archivo capturas_datos.sql en el Escritorio. Cópialo también a la memoria USB y llévalo al portátil (por ejemplo a C:\Capturas).

3.2 — Restaurar la copia (en el PORTÁTIL)

Abre cmd en el portátil. Primero crea la base de datos vacía (te pedirá TU_CONTRASEÑA del portátil):

"C:\Program Files\PostgreSQL\17\bin\createdb.exe" -U postgres capturas

Después restaura los datos:

"C:\Program Files\PostgreSQL\17\bin\psql.exe" -U postgres -d capturas -f "C:\Capturas\capturas_datos.sql"
¿Prefieres una demo "en limpio" sin tus datos reales? Entonces sáltate la Parte 3 entera. Más abajo, en la Parte 5, te indico cómo crear las tablas vacías y unos datos de ejemplo en su lugar.

Parte 4 · Configurar la conexión en el portátil

  1. En la carpeta C:\Capturas del portátil, busca el archivo .env (si no se ve, activa "Elementos ocultos" en la pestaña Vista del Explorador).
  2. Ábrelo con el Bloc de notas (clic derecho → Abrir con → Bloc de notas).
  3. Busca la primera línea, DATABASE_URL=..., y déjala exactamente así (cambiando TU_CONTRASEÑA por la del portátil):
DATABASE_URL="postgresql://postgres:TU_CONTRASEÑA@localhost:5432/capturas?schema=public"

Guarda el archivo (Archivo → Guardar) y ciérralo.

Las líneas de WATCH_FOLDER, GASTOS_FOLDER, etc. apuntan a carpetas de tu Dropbox que no existen en el portátil. Para la demo no importan (no vamos a importar PDFs nuevos), puedes dejarlas como están.

Parte 5 · Instalar y arrancar en el portátil

1) Abre cmd y entra en la carpeta del proyecto:

cd C:\Capturas

2) Instala las dependencias (tarda unos minutos, descarga muchas cosas):

npm install

3) Genera el cliente de base de datos:

npx prisma generate
Solo si te SALTASTE la Parte 3 (demo sin datos reales): crea las tablas vacías y unos datos de ejemplo ejecutando, uno por uno: 
npm run db:push
npm run prisma:seed
npm run seed:sailors
Si SÍ restauraste tus datos en la Parte 3, NO ejecutes estos tres comandos.

4) Arranca la aplicación:

npm run dev

Espera a que ponga algo como Ready in ... - Local: http://localhost:3000. Deja esta ventana negra abierta mientras hagas la demo (si la cierras, la app se apaga).

5) Abre el navegador del portátil en:

http://localhost:3000

Apagar y volver a encender

Para apagar: cierra la ventana negra del npm run dev (o pulsa Ctrl+C dentro de ella).

Para volver a encender otro día: abre cmd, escribe cd C:\Capturas, pulsa Enter, escribe npm run dev, pulsa Enter, y abre http://localhost:3000 en el navegador. (No hace falta volver a instalar nada.)

Recuerda: esta copia del portátil es independiente. Lo que cambies aquí NO afecta a tu ordenador principal, y viceversa. Es perfecto para demos sin riesgo.
